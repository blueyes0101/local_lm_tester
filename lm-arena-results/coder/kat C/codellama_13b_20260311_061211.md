This involves keeping track of the last time each refresh token was used and setting a TTL for the token. 6. Token blacklisting and token rotation are two different approaches to invalidating tokens after they have been used.

Token blacklisting involves keeping a list of tokens that are no longer valid and checking this list when a user attempts to authenticate. If a token is found on the blacklist, the user is not allowed to authenticate and must provide a new token. Token blacklisting is useful when the risk of token theft is high, as it allows for immediate revocation of tokens.

Token rotation, on the other hand, involves rotating the token every time it is used. This means that each time a user logs in, a new token is issued and the old token is invalidated. Token rotation is useful when the risk of token theft is low, as it reduces the number of tokens that are valid at any given time.

In general, token rotation is a more secure approach than token blacklisting, as it reduces the number of tokens that are valid at any given time. However, it requires more computational resources to issue new tokens every time a user logs in. Token blacklisting is a more efficient approach, but it may allow for a longer period of time before a token is invalidated.

In summary, the choice between token blacklisting and token rotation will depend on the specific requirements of your application and the level of risk you are willing to accept. If the risk of token theft is high, token rotation may be the better choice. If the risk is low, token blacklisting may be the better choice.

import { useMemo } from 'react';

interface DataTableProps<T> {
  columns: Array<{
    key: string;
    header: string;
    width: string;
    sortable: boolean;
    renderCell?: (row: T) => ReactNode;
  }>;
  data: Array<T>;
  isControlled?: boolean;
  pageSize?: number;
  selectable?: boolean;
}

const DataTable = <T,>({ columns, data, isControlled, pageSize, selectable }: DataTableProps<T>) => {

  useEffect(() => {
    setSelectedRows([]);
  }, [data]);

    setSelectedRows(checked ? data.map((row) => row) : []);

    const checked = event.target.checked;
    setSelectedRows((prevSelectedRows) =>
      checked ? [...prevSelectedRows, row] : prevSelectedRows.filter((r) => r !== row),
    );
  };

    if (sort.key === key) {
      setSort({ key, direction: sort.direction === 'asc' ? 'desc' : 'none' });
    } else {
      setSort({ key, direction: 'asc' });
    }
  };

  const handlePageChange = (page: number) => {
    setSelectedRows([]);
  };

  const handlePageSizeChange = (size: number) => {
    setSelectedRows([]);
  };

  const memoizedData = useMemo(() => data.map((row) => ({ ...row, selected: selectedRows.includes(row) })), [
    data,
    selectedRows,
  ]);

  return (
    <Table>
      <TableHead>
        <TableRow>
          {columns.map((column, index) => (
            <TableCell
              key={`${column.key}-${index}`}
              align={column.align}
